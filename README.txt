CRUD-PRUEBA TECNICA 

El presente aplicativo se consolido con el fin de evidenciar capacidades tecnicas de desarrollo,
se utilizo Angular, bootstrap y firebase. En el aplicativo se puede crear un registro, modificarlo
, rebisar y eliminar.


PRE-REQUISITOS

Se debe contar con VScode y Angular


INSTALAR

Ingresando en siguiente comando se añade las funcionalidades necesarias para utilizar Firebase

npm install firebase@7.24.0 @angular/fire@6.0.3 --save


PRUEBAS

ejecutar el aplicación

ng serve

abrir en el navegador web localhost:4200

puede ingesar nuevos registros en la base de datos, observar un listado y actualizar un 
registro existente


CONSTRUIDO CON

- Angular
